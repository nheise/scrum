## Scrum Scheduler

In the project directory, you can build:

### `mvn clean package`

and run:

### `java -jar target/scrum-0.0.1-SNAPSHOT.jar`

Runs the app in the development mode.<br />
[http://localhost:8080](http://localhost:8080).


### API endpoints
